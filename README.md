# 🐍 SnakeGPT AI

> A professional‑grade AI code assistant and OSINT intelligence hub powered by **LLaMA 3.3 70B** via the Groq inference API.

---

## 🎯 Overview

SnakeGPT AI is a web‑based coding assistant that lets you:

- **Write, debug, explain, and optimise** code in any language.
- **Live‑preview** rendered HTML, CSS, and JavaScript right next to the chat pane.
- **Store chat histories** locally with rename/delete functionality.
- **Use it from any device** – fully responsive UI.
- **Tap into LLaMA 3.3 70B** via Groq for ultra‑fast inference.

---

## ✨ Features

| Feature | What it does |
|---|---|
| 🤖 AI Code Assistant | Generate code, fix bugs, refactor and explain in 30+ languages |
| 🖥️ Live HTML Preview | Split‑screen panel renders HTML/CSS/JS instantly |
| 💬 Chat History | Persisted locally; rename and delete chats |
| 🕵️ OSINT Toolkit | Built‑in OSINT commands (e.g., network reconnaissance, dark‑web search) |
| 📱 Mobile‑First | Responsive UI that works on phones, tablets, and desktops |
| ⚡ Speed | Groq's low‑latency inference API guarantees sub‑second responses |

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python 3.8 + Flask 3.0 |
| AI | LLaMA 3.3 70B via Groq |
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Syntax Highlighting | highlight.js |
| Markdown | marked.js |
